# Sustain-Arduino

## Getting started
### Things you need
<a href="https://www.pjrc.com/teensy/td_download.html">Teensyduino</a> (latest version)<br>
<a href="https://www.arduino.cc/en/Main/Software">Arduino Studio</a> the latest version of Arduino Studio may not yet be supported by Teensyduino. So pick one that is.<br>
The Arduino libraries included in the libs folder of this repository.<br>
<br>
### Installation
Install Arduino Studio (run the setup).<br>
Install Teensyduino (run the setup).<br>
Using Arduino Studio, install all the includes libraries (Sketch -> Include Library -> Add .ZIP Library).<br>
